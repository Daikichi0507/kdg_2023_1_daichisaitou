READ MEこのサイトについて

どんなサイトなの？

このサイトはさまざまなバスケット選手を選択して、自分の思うバスケット選手のいい点やダメな点を自分で評価することが出来るサイトです。

このサイトの目的

このサイトを作った目的は周りの意見を聞いて評価する人がこの世の中にはたくさんあるような気がしたので、
それなら自分の思う意見を出すことが出来るサイトを作ればいいのではないかと思い、このサイトを作ることにしました。

このサイトを作った理由

このサイトを作った理由として、バスケを知らない方々に向けて
どのような選手がいてその選手はどのくらい強くて、どのくらい上手いのかというのを知って欲しくてこのサイトを作りました。
